
# 递推序列解析延拓 — 工程化数值实验方案

## 一、项目概述

### 1.1 问题背景

对于具有递推结构的离散数列 $a_n$，能否将其"平滑地"推广到非整数的实数（甚至复数）取值？这类问题在数学上称为**解析延拓（Analytic Continuation）**。经典案例如 Gamma 函数 $\Gamma(z)$ 将阶乘 $n!$ 延拓至复平面。

本项目关注一类更极端的情况：增长极快的递推数列（如迭代幂次 Tetration），其泛函方程具有高度非线性，延拓不再唯一。我们需要通过数值方法，在多种可能的延拓中寻找最"自然"的解。

### 1.2 项目目标

1. **通用框架**：构建可扩展的 Python 工程，支持多种递推序列和多种数值方法
2. **数值求解**：对给定递推序列，计算其在非整数点处的解析延拓值
3. **最优化**：在多种合法延拓中，寻找使应变能（曲率积分）最小的"最自然"解
4. **对比分析**：系统评估不同方法在精度、效率、收敛性上的表现

### 1.3 适用范围

- 数值方法仅限经典数值分析工具（样条、谱方法、变分法等），**不涉及 AI/神经网络方法**
- 优先处理实数域延拓，复平面延拓作为进阶方向
- 序列类型覆盖四类（见 §2），架构支持自定义扩展

## 二、数学框架

### 2.1 统一建模：泛函方程

将离散递推 $a_n = g(n, a_{n-1})$ 连续化为泛函方程：

$$f(z) = g\big(z,\, f(z-1)\big), \quad f(n_0) = a_{n_0}$$

其中 $n_0$ 为初始整数点。在基底区间 $[n_0, n_0+1]$ 上构造插值函数 $f(z)$，然后通过正向递推 $f(z) = g(z, f(z-1))$ 或反向递推 $f(z-1) = g^{-1}(z, f(z))$ 向两侧延拓。

### 2.2 四类目标递推序列

#### 类型 A：变底数迭代幂次

$$a_n = n^{(n-1)^{(n-2)^{\cdots^{2^1}}}}$$

- **递推关系**：$a_n = n^{a_{n-1}}$，$a_1 = 1$
- **泛函方程**：$f(z) = z^{f(z-1)}$，即 $g(z, w) = z^w$
- **反向递推**：$f(z-1) = \frac{\ln f(z)}{\ln z}$
- **边界条件**：$f(1) = 1$，$f(2) = 2$
- **基底区间**：$[1, 2]$

#### 类型 B：固定底数迭代幂次

$$a_n = b^{b^{\cdot^{\cdot^{b}}}} \quad (\text{$n$ 层})$$

- **递推关系**：$a_n = b^{a_{n-1}}$，$a_1 = b$
- **泛函方程**：$f(z) = b^{f(z-1)}$，即 $g(z, w) = b^w$（$g$ 不显含 $z$）
- **反向递推**：$f(z-1) = \log_b f(z)$
- **收敛条件**：$e^{-e} \leq b \leq e^{1/e}$ 时无限塔收敛
- **边界条件**：$f(1) = b$，$f(2) = b^b$
- **基底区间**：$[1, 2]$
- **特殊性**：底数恒定，可参考 Kneser 构造法

#### 类型 C：阶乘类递推

$$a_n = n! = n \cdot (n-1) \cdot \cdots \cdot 1$$

- **递推关系**：$a_n = n \cdot a_{n-1}$，$a_1 = 1$
- **泛函方程**：$f(z) = z \cdot f(z-1)$，即 $g(z, w) = z \cdot w$
- **反向递推**：$f(z-1) = f(z) / z$
- **标准解**：$\Gamma(z+1)$（对数凸性保证唯一性，Bohr-Mollerup 定理）
- **基底区间**：$[1, 2]$
- **验证价值**：作为有已知解析解的"ground truth"案例，用于验证算法正确性

#### 类型 D：自定义递推

$$f(z) = g\big(z, f(z-1)\big)$$

用户通过注册函数 $g(z, w)$ 和初始条件 $f(n_0) = v_0$ 来定义任意递推关系。系统要求：

- $g(z, w)$ 在基底区间上连续且对 $w$ 严格单调（保证反向递推有唯一解）
- 提供 $g$ 的显式反函数或数值求根方式

### 2.3 边界条件的导数约束

对泛函方程 $f(z) = g(z, f(z-1))$ 两端关于 $z$ 求导，利用链式法则：

$$f'(z) = g_z(z, f(z-1)) + g_w(z, f(z-1)) \cdot f'(z-1)$$

代入基底区间的边界 $z = n_0 + 1$：

$$f'(n_0+1) = g_z(n_0+1, f(n_0)) + g_w(n_0+1, f(n_0)) \cdot f'(n_0)$$

这给出了 $f'(n_0+1)$ 与 $f'(n_0)$ 之间的**线性约束关系**。对各类序列的具体形式：

| 序列类型 | $g(z,w)$ | $g_z$ | $g_w$ | $f'(n_0+1)$ 约束 |
|---------|----------|-------|-------|------------------|
| A: 变底数幂次 | $z^w$ | $z^w \frac{w}{z}$ | $z^w \ln z$ | $f'(2) = 2f'(1)\ln 2 + 1$ |
| B: 固定底数 | $b^w$ | $0$ | $b^w \ln b$ | $f'(2) = b^b \ln b \cdot f'(1)$ |
| C: 阶乘类 | $zw$ | $w$ | $z$ | $f'(2) = 1 + 2f'(1)$ |

通过连续求高阶导数，可以用 Faà di Bruno 公式（贝尔多项式）递推地生成任意 $K$ 阶导数的约束关系，复杂度 $\mathcal{O}(K^2)$。

### 2.4 应变能与"自然性"度量

在基底区间 $[n_0, n_0+1]$ 上，定义能量泛函：

$$J(f) = \int_{n_0}^{n_0+1} \big[f''(z)\big]^2 dz$$

物理含义：$f''(z)$ 是曲线的曲率，$J(f)$ 是总弯曲能量。"最自然"的延拓对应 $J(f)$ 的极小化。对于有唯一解析解的序列（如阶乘），该极小值应与已知解吻合。

更一般的能量泛函（可选）：

$$J_k(f) = \int_{n_0}^{n_0+1} \big[f^{(k)}(z)\big]^2 dz, \quad k = 2, 3, \ldots$$

## 三、数值方法体系

### 3.1 方法总览

| 方法 | 精度阶 | 自由度参数 | 时间复杂度 | 适用场景 |
|------|--------|-----------|-----------|---------|
| M1: 三次 Hermite 样条 | $\mathcal{O}(h^4)$ | 1（$f'(n_0)=c$） | $\mathcal{O}(1)$ | 快速基线估算 |
| M2: 五次 Hermite 样条 | $\mathcal{O}(h^6)$ | 2（$f', f''$） | $\mathcal{O}(1)$ | 中等精度 |
| M3: 切比雪夫谱方法 | 指数收敛 | $N$ 个谱系数 | $\mathcal{O}(N^3)$ | 高精度逼近 |
| M4: 变分样条（分段高阶） | 可调 | 多段多项式系数 | $\mathcal{O}(M^3)$ | 复杂拓扑 |
| M5: 正则化迭代法 | 可调 | 正则化参数 $\lambda$ | $\mathcal{O}(N_{\text{iter}} \cdot N^2)$ | 自适应平滑 |

### 3.2 M1：三次 Hermite 样条法（基线方法）

**核心思想**：在 $[n_0, n_0+1]$ 上构造三次多项式 $P(z) = a_0 + a_1 z + a_2 z^2 + a_3 z^3$，匹配两端函数值和一阶导数值。

**输入参数**：
- 自由度 $c = f'(n_0)$
- 由 §2.3 公式计算 $f'(n_0+1)$

**算法步骤**：
1. 以 $P(n_0) = f(n_0)$、$P(n_0+1) = f(n_0+1)$、$P'(n_0) = c$、$P'(n_0+1) = h(c)$ 建立 $4\times 4$ 线性方程组
2. 求解得到 $(a_0, a_1, a_2, a_3)$
3. 计算基底区间内任意点的值：$f(z) = P(z)$
4. 利用反向递推 $f(z-1) = g^{-1}(z, f(z))$ 向 $z < n_0$ 延拓

**应变能优化**：
由于三次多项式的 $P''(z)$ 是一次函数，能量 $J(c)$ 是 $c$ 的严格凸二次函数。令 $dJ/dc = 0$ 可**解析地**求出最优 $c^*$，无需迭代。

**复杂度**：$\mathcal{O}(1)$（常数量级的浮点运算）

**局限**：
- 仅保证 $\mathcal{C}^1$ 连续
- 基底区间内部精度有限（$\sim 10^{-3}$ 量级）
- 不适合匹配高阶导数约束

### 3.3 M2：五次 Hermite 样条法

**核心思想**：将多项式阶数提升到 5 次，同时匹配函数值、一阶和二阶导数，共 6 个约束条件。

**输入参数**：
- 自由度 $c_1 = f'(n_0)$、$c_2 = f''(n_0)$
- 由 §2.3 和二阶导数递推公式计算 $f'(n_0+1)$ 和 $f''(n_0+1)$

**算法步骤**：同 M1，但方程组变为 $6 \times 6$。

**应变能优化**：$J(c_1, c_2)$ 是 $(c_1, c_2)$ 的严格凸二次函数，通过求解 $2 \times 2$ 线性系统 $\nabla J = 0$ 得到全局最优解。

**复杂度**：$\mathcal{O}(1)$

**优势**：保证 $\mathcal{C}^2$ 连续，精度显著提升。

### 3.4 M3：切比雪夫谱方法（高精度方法）

**核心思想**：在基底区间 $[n_0, n_0+1]$ 上将 $f(z)$ 展开为 Chebyshev 级数：

$$f(z) \approx \sum_{k=0}^{N} a_k T_k(2(z - n_0) - 1)$$

其中 $T_k$ 为第 $k$ 阶 Chebyshev 多项式。通过变分原理，将应变能极小化问题转化为带约束的二次规划（QP）。

**算法步骤**：

**Step 1 — 构建离散化框架**
- 设定截断阶数 $N$（根据目标精度 $p$ 选择 $N \sim \alpha p$，$\alpha$ 为经验系数）
- 利用 Chebyshev-Gauss-Lobatto 节点 $z_j = n_0 + \frac{1}{2}(1 + \cos\frac{j\pi}{N})$，$j=0,\ldots,N$

**Step 2 — 构建微分矩阵**
- Chebyshev 微分矩阵 $\mathbf{D}$ 将函数值映射为导数值：$\mathbf{f}' = \mathbf{D} \mathbf{f}$
- 二阶微分矩阵 $\mathbf{D}_2 = \mathbf{D}^2$
- 能量泛函离散化为：$J(\mathbf{a}) = \mathbf{a}^T \mathbf{M} \mathbf{a}$，其中 $\mathbf{M}$ 为正定矩阵

**Step 3 — 建立约束条件**
- 边界值约束：$f(n_0) = v_0$，$f(n_0+1) = v_1$（线性约束）
- 导数约束：由 §2.3 的 $K$ 阶导数递推关系生成的 $K$ 个线性等式
- 约束矩阵形式：$\mathbf{C} \mathbf{a} = \mathbf{b}$

**Step 4 — 求解 QP 问题**
$$\min_{\mathbf{a}} \mathbf{a}^T \mathbf{M} \mathbf{a} \quad \text{s.t.} \quad \mathbf{C} \mathbf{a} = \mathbf{b}$$

引入拉格朗日乘子 $\boldsymbol{\lambda}$，等价于求解线性系统：

$$\begin{pmatrix} 2\mathbf{M} & \mathbf{C}^T \\ \mathbf{C} & \mathbf{0} \end{pmatrix} \begin{pmatrix} \mathbf{a}^* \\ \boldsymbol{\lambda}^* \end{pmatrix} = \begin{pmatrix} \mathbf{0} \\ \mathbf{b} \end{pmatrix}$$

规模 $(N + K + 1) \times (N + K + 1)$，稠密求解复杂度 $\mathcal{O}((N+K)^3)$。

**Step 5 — 延拓计算**
- 求值：$f(z) = \sum a_k^* T_k(\cdot)$
- 反向递推向 $z < n_0$ 延拓

**收敛特性**：若真实解 $f(z)$ 在基底区间上解析，则误差 $\|f - f_N\| = \mathcal{O}(e^{-\alpha N})$（谱收敛/指数收敛）。

**复杂度**：$\mathcal{O}(N^3)$ 时间，$\mathcal{O}(N^2)$ 空间。

### 3.5 M4：变分样条法（分段高阶）

**核心思想**：将基底区间 $[n_0, n_0+1]$ 划分为 $M$ 个子区间，在每个子区间上使用 $p$ 次多项式，整体通过变分法极小化能量泛函。

**适用场景**：当延拓函数在基底区间上可能有局部结构变化时（如某些自定义递推），单一全局多项式表现不佳。

**算法步骤**：
1. 将 $[n_0, n_0+1]$ 均匀剖分为 $M$ 段
2. 每段使用 $p$ 次多项式，共 $M(p+1)$ 个自由度
3. 施加 $\mathcal{C}^{p-1}$ 内部连续性约束
4. 施加边界值和导数约束
5. 极小化 $J(f)$（二次规划）
6. 自适应 $h$-refinement：在残差较大的子区间加密

**复杂度**：$\mathcal{O}(M^3 p^3)$，$M$ 和 $p$ 可根据误差估计自适应调整。

### 3.6 M5：正则化迭代法

**核心思想**：不从构造插值出发，而是直接在泛函方程残差和能量泛函之间做权衡，通过迭代逐步逼近最优解。

**算法步骤**：
1. 在基底区间上离散化 $f(z)$ 为 $N$ 个节点值
2. 定义混合目标函数：$\mathcal{L}(f) = \lambda \cdot J(f) + \sum_i \text{residual}_i^2$
3. 其中残差来自泛函方程在内部节点的违反程度
4. 用梯度下降或 L-BFGS 优化 $\mathcal{L}(f)$
5. 正则化参数 $\lambda$ 控制"方程满足度"与"平滑度"的权衡

**优势**：不要求 $f$ 严格满足泛函方程（允许近似满足），对病态问题更鲁棒。

**复杂度**：$\mathcal{O}(N_{\text{iter}} \cdot N^2)$

### 4.2 核心接口设计

#### 序列注册接口

```python
from abc import ABC, abstractmethod
from dataclasses import dataclass

@dataclass
class RecurrenceSequence(ABC):
    """递推序列的统一抽象"""
    name: str
    n0: float                          # 基底区间左端点
    f_n0: float                        # f(n0) 的值
    f_n0_plus_1: float                 # f(n0+1) 的值

    @abstractmethod
    def g(self, z: float, w: float) -> float:
        """正向递推函数 g(z, f(z-1))"""
        ...

    @abstractmethod
    def g_inv(self, z: float, f_val: float) -> float:
        """反向递推：已知 f(z)，求 f(z-1)"""
        ...

    @abstractmethod
    def g_z(self, z: float, w: float) -> float:
        """g 对 z 的偏导数"""
        ...

    @abstractmethod
    def g_w(self, z: float, w: float) -> float:
        """g 对 w 的偏导数"""
        ...

    def derivative_constraint(self, deriv_at_n0_plus_1: list[float]) -> list[float]:
        """
        给定 f^(k)(n0)（k=0,1,...,K），递推计算 f^(k)(n0+1)。
        默认使用一阶链式法则，子类可覆盖以支持高阶。
        """
        ...
```

#### 数值方法接口

```python
from abc import ABC, abstractmethod
from dataclasses import dataclass
import numpy as np

@dataclass
class ContinuationResult:
    """解析延拓结果"""
    method_name: str
    sequence_name: str
    optimal_params: dict               # 最优自由度参数
    strain_energy: float               # 应变能 J(f)
    eval_at: dict[float, float]        # {z: f(z)} 各点求值
    basis_coefficients: np.ndarray      # 基底函数系数
    metadata: dict                      # 其他信息（耗时、迭代次数等）

class ContinuationMethod(ABC):
    """数值方法的统一抽象"""
    name: str

    @abstractmethod
    def solve(self, seq: RecurrenceSequence,
              target_points: list[float],
              **kwargs) -> ContinuationResult:
        """
        在 seq 定义的递推关系下，计算 target_points 处的解析延拓值。
        """
        ...

    @abstractmethod
    def evaluate(self, z: float, result: ContinuationResult) -> float:
        """给定已求解的结果，在任意点 z 处求值"""
        ...
```

## 五、实验流程

### 5.1 实验一：基线对比实验

**目标**：用所有方法计算四类序列在关键点处的延拓值，建立性能基线。

**步骤**：
1. 对每类序列，分别用 M1–M5 方法求解
2. 计算目标点集：$z \in \{0.5, 1.5, 2.5, 3.5\}$
3. 记录每个方法在每类序列上的：
   - 各点函数值 $f(z)$
   - 应变能 $J(f)$
   - 计算耗时
4. 对类型 C（阶乘），以 $\Gamma(z+1)$ 为标准计算相对误差
5. 生成对比表格和折线图

**关键参数**：
- M1: $c$ 通过解析优化自动确定
- M2: $(c_1, c_2)$ 通过 $2\times2$ 系统优化
- M3: $N = 16, 32, 64$（三档截断阶数），$K = 2$（匹配导数阶数）
- M4: $M = 4, 8$，$p = 3, 5$
- M5: $\lambda = 10^{-2}, 10^{-4}, 10^{-6}$，$N = 50$ 节点

**输出**：`results/exp_baseline/` 下的 CSV 表格和 PNG 图表

### 5.2 实验二：收敛性分析实验

**目标**：验证各方法的收敛阶和精度极限。

**步骤**：
1. 以类型 C（阶乘）为测试对象，真实解 $\Gamma(z+1)$ 已知
2. 逐步提高各方法的精度参数（$N$、$p$、$M$ 等）
3. 在 $z = 1.5$ 处计算与 $\Gamma(2.5)$ 的绝对误差
4. 绘制"自由度 vs. 误差"的双对数图（log-log plot）
5. 对 M3（Chebyshev）验证指数收敛；对 M1/M2 验证多项式收敛阶
6. 额外：对类型 A、B，增大 $N$ 观察结果是否稳定（无 ground truth 时的自洽性检验）

**评估指标**：
- 绝对误差：$|f_N(z) - f_{\text{true}}(z)|$
- 收敛速率：$\log(\epsilon_{N}/\epsilon_{2N}) / \log 2$
- 应变能的收敛值：$J(f_N) \to J^*$？

**输出**：`results/exp_convergence/` 下的收敛曲线图和拟合数据

### 5.3 实验三：应变能对比实验

**目标**：对比各方法找到的应变能极小值，分析不同方法的"自然性"差异。

**步骤**：
1. 对每类序列，用 M3（Chebyshev，$N=64$）的结果作为"高精度参考"
2. 计算各方法得到的 $J(f)$，与参考值比较
3. 对类型 A（变底数迭代幂次），扫描 $f'(1) = c$ 的完整参数空间，绘制 $J(c)$ 曲线
4. 验证 $J(c)$ 确实是严格凸的二次函数（M1 情况下）
5. 绘制不同 $c$ 值对应的延拓函数曲线族

**输出**：`results/exp_energy/` 下的能量曲线图和最优参数表

### 5.4 实验四：延拓域扩展实验

**目标**：测试基底区间外的延拓质量。

**步骤**：
1. 以 M3（$N=64$）在 $[1,2]$ 上求解基底函数
2. 正向递推至 $z \in [2, 5]$，在每个整数点 $z = n$ 处检验 $f(n) = g(n, f(n-1))$ 是否精确满足
3. 反向递推至 $z \in [0, 1]$，计算 $f(0) = f'(1)$ 并与理论预期对比
4. 尝试反向递推至 $z = -1$，验证 $f(-1) = 0$（类型 A 情况）
5. 记录反向递推的数值稳定性（条件数、误差放大）

**输出**：`results/exp_extension/` 下的延拓曲线图和误差分析

## 六、评估指标体系

### 6.1 精度指标

| 指标 | 定义 | 适用场景 |
|------|------|---------|
| 绝对误差 $\epsilon_{\text{abs}}$ | $|f_{\text{computed}}(z) - f_{\text{true}}(z)|$ | 有 ground truth（类型 C） |
| 递推残差 $\epsilon_{\text{recur}}$ | $|f(z) - g(z, f(z-1))|$ 在整数点的残差 | 所有类型 |
| 应变能 $J$ | $\int [f'']^2 dz$ | 衡量"自然性" |
| 能量比 $J/J_{\text{ref}}$ | 与 Chebyshev 参考解的比值 | 方法对比 |

### 6.2 效率指标

| 指标 | 含义 |
|------|------|
| 墙钟时间 $T$ | 单次求解耗时（ms） |
| 精度/时间比 | $-\log_{10} \epsilon / T$ |
| 内存峰值 | 最大内存占用（MB） |

### 6.3 鲁棒性指标

| 指标 | 含义 |
|------|------|
| 条件数 | 基底区间求解的矩阵条件数 |
| 反向递推稳定性 | 向 $z < n_0$ 延拓时的误差增长速率 |
| 参数敏感性 | $c$ 偏离最优值时 $J(c)$ 的增长斜率 |

### 8.2 关键验证点

1. **正确性验证**：类型 C（阶乘）的所有方法结果必须收敛到 $\Gamma(z+1)$
2. **唯一性验证**：不同方法对同一序列的延拓在增大精度后应收敛到同一值
3. **能量极小性验证**：$J(f_{\text{optimal}}) \leq J(f)$ 对所有合法 $f$ 成立
4. **奇点验证**：类型 A 在 $z = -2$ 处发散（与理论分析一致）

## 九、风险与对策

| 风险 | 影响 | 对策 |
|------|------|------|
| Chebyshev QP 矩阵病态 | 高 $N$ 时数值不稳定 | 使用正交化的基函数；Tikhonov 正则化 |
| 反向递推误差爆炸 | $z \ll n_0$ 时结果不可靠 | 使用高精度算术（`mpmath`）；限制延拓范围 |
| 自定义序列的反函数无解析解 | M1/M2 的反向递推失效 | 提供 `scipy.optimize.brentq` 数值求根的备选路径 |
| 应变能极小值不唯一 | 多个局部极小 | 利用凸性证明（对三次/五次样条可证唯一）；对 Chebyshev 使用全局优化 |
